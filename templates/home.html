Available apps are:

  1. Language Detector: This is a simple app which uses Python library langdetect to detect what language a given text belongs to. The model is quite accuracte if the sample text is large enough. Currently, supported languages are:

    [{{ languages }}]

  2. Image Tools: This app provides some basic functionalities for JPG and PNG images, like compression, resizing, and converting format. Under the hood, the app uses the Pillow library.
  3. Youtube Song Downloader: As the name suggests, this app downloads song from YouTube using its URL. The app gives options to download the audio file with various qualities, in terms of kilo bits per second. I use another library pytube in the back-end.
  4. Machine Learning Models: This app contains some of the machine learning and deep learning models.
